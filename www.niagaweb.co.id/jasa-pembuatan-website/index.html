FAQ Seputar Jasa Pembuatan
Website Niagaweb

Apa itu Website Company Profile?

Website company profile adalah jenis website yang bertujuan untuk mengenalkan profile sebuah perusahaan beserta produk maupun jasa yang ditawarkan perusahaan. Kami Niagaweb melayani jasa pembuatan website company profile dengan berbagai macam pilihan desain website.

Apa itu website Toko Online / Online Shop?

Website Toko Online / Online Shop adalah jenis website yang lebih spesifik untuk menjual produk maupun jasa identik dengan adanya fitur fitur untuk belanja di website. Kami Niagaweb juga melani jasa desain website Toko Online / Online Shop untuk Anda.

Berapa biaya pembuatan website di Niagaweb?

1. Biaya pembuatan landingpage mulai dari Rp. 760.000
2. Biaya pembuatan website perusahaan / company profile mulai dari Rp. 1.500.000 
3. Biaya pembuatan website toko online / online shop mulai dari Rp 2.500.000 
Apabila Anda membutuhkan plugin tambahan bisa dikenakan biaya tambahan tentunya Anda dapat konsultasi terlebih dahulu dengan tim Kami.

Apa saja yang harus disiapkan jika saya ingin memesan jasa pembuatan website?

Dalam layanan jasa pembuatan website ini, kami sudah menyediakan template yang siap dipakai. Yang perlu Anda siapkan di antaranya adalah:

1. Nama domain website

2. Konten atau artikel yang akan di publish di website

3. Gambar-gambar yang akan di publish

Apakah biaya jasa desain website sudah termasuk Domain dan Hosting?

Untuk paket jasa website company profile dan online shop sudah termasuk domain dan hosting untuk satu tahun pertama. Untuk ketentuan lebih lanjut Anda bisa konsultasi terlebih dahulu dengan kami.

Saya sudah punya hosting dan domain selain dari Niagaweb. Apakah Niagaweb tetap bisa membuatkan website untuk saya ?

Bisa! Kami bisa bantu membuatkan website meskipun Anda sudah memiliki hosting dan domain di layanan selain Niagaweb. Hanya saja, jika Anda ingin kami bantu kelola website Anda pada tahun berikutnya, maka Anda harus tetap melakukan request pengelolaan website secara terpisah di layanan kami.
